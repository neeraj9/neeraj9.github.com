Setup Markdoc

I have been using Markdoc for some time and found its clean configuration and generated html interface very appealing. Apart from that Markdoc is written in one of my favourite language - python.

This page talks about installing Markdoc and using it to generate static website which you could later publish to Apache/Nginx or any other webserver.

How to Setup on CentOS

Although this activity was performed on CentOS 6.1, still the information is valid for any CentOS version 6.1+.

The first step is to install all the dependencies via the yum package installer.

$ sudo yum install python-pip python-cherrypy2 python-jinja2 python-pygments PyYAML.x86_64 python-webob1.0 
$ sudo pip-python install Markdoc

After you have executed the install of Markdoc, you will see the following output on the screen. This indicates that the installation was successful.

Successfully installed argparse Markdoc Markdown Cleaning up...

Now you can use Markdoc via the command /usr/bin/markdoc.
