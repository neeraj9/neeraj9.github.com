Update Intel Microcode in Linux

I bought a great piece of hardware - Nehelam i7 and assembed other pieces to create a rock solid server, but suddenly I realized that the CPU didnt have fixes which it should have. This is where microcode patching comes into play and since I am using Linux, so doing it is a charm.

Lets take a look how its done everytime the system boots up (since non-BIOS microcode update are not persistent).

The Process

Download the latest microcode from intel website and install it.

# Install the microcode_ctl which does patching for us.
$ sudo yum install microcode_ctl
# download the latest microcode (download new if this one is old now)
$ wget http://downloadmirror.intel.com/20728/eng/microcode-20111110.tgz
$ tar -xvzf microcode-20111110.tgz
$ sudo cp microcode.dat /lib/firmware/microcode.dat
$ sudo microcode_ctl -u

Wow! Thats it.

Enjoy!
